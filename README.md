# SQL Business Analysis (Chinook)

## Overview
This project analyzes transactional business data using SQL to identify revenue trends, customer value, and product performance. It demonstrates practical data analytics skills such as multi-table joins, aggregations, and business insight generation.

## Dataset
Chinook sample database representing a digital media store with customers, invoices, and product purchases.

## Key Questions Answered
- How does revenue trend over time?
- Who are the highest spending customers?
- Which countries generate the most revenue?
- What products generate the most sales?
- What is the average order value?

## Key Findings
- Total revenue analyzed: $2328.60
- Total orders: 412
- Average order value: $5.65
- Top revenue country: USA ($523.06)
- Highest spending customer: Helena Holý ($49.62)

## Skills Demonstrated
- SQL JOIN operations
- Aggregations (SUM, AVG, COUNT)
- GROUP BY and ORDER BY queries
- Business data analysis

## Project Structure
- sql/analysis.sql → SQL queries
- outputs/results.txt → query results
- data.db → database file
